## 旧模块
旧模块总是会频繁输出
```
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]
[128, 128, 128, 128, 128]

```

> 可能是由于接到了3v的vcc口上，供电不足导致。


* 另外解释不通的是为什么模块用外置电源供电就不行，必须得用树莓派引脚的vcc供电。

> 因为要和检测模块树莓派组成回路，即时钟总线和数据总线也只是输出高地电频，需有回路才能检测的到。
